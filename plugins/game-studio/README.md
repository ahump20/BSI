# Game Studio

Claude-native port of the OpenAI-curated `game-studio` bundle.

## Coverage

- Browser-game routing and stack selection
- Shared architecture for simulation, rendering, UI, and save boundaries
- Phaser-first 2D implementation guidance
- Three.js and React Three Fiber 3D guidance
- 3D asset shipping and 2D sprite workflows
- Browser-game HUD, overlay, and menu design
- Structured browser-game playtesting and QA

## Included Skills

- `game-studio`
- `web-game-foundations`
- `game-ui-frontend`
- `phaser-2d-game`
- `react-three-fiber-game`
- `three-webgl-game`
- `web-3d-asset-pipeline`
- `sprite-pipeline`
- `game-playtest`

## Notes

Browser QA in this plugin pairs well with the already-enabled Anthropic Playwright plugin, but Playwright is not bundled here.

## Representative Prompts

- `Help me choose the right browser-game stack for a 2D tactics prototype.`
- `Set up a Phaser architecture for a top-down action game with DOM HUD overlays.`
- `Build the Three.js runtime for this exploration prototype and keep the first playable screen low-chrome.`
- `Prepare these GLB assets for browser delivery and trim the runtime cost.`
- `Run a playtest pass on this game and report the blocking issues in severity order.`

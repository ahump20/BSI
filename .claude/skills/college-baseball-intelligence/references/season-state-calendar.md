# Season-State Calendar

## Phase Map

Preseason (Nov-Jan): Roster shape reliable. Nothing statistical.
Early non-conf (Feb Wk 1-3): BB%, K%, command reliable. ERA, BABIP, power are noise.
Conference start (Mar Wk 4-8): Conference rate stats, run diff reliable. Individual BABIP, FIP need more IP.
Midseason (Apr Wk 9-12): Most rates stabilized (150+ PA, 40+ IP). RISP still noisy.
Stretch run (May Wk 13-16): Everything reliable. Arm count is critical.
Postseason (Selection-CWS): Matchup-specific. Arm availability. Season aggregates are background only.

## Stabilization Thresholds

BB%, K%: 60 PA (Week 3-4) | K/BB pitcher: 25 IP (Week 4-5) | wOBA: 150 PA (Week 6-8)
wRC+: 200 PA (Week 8-10) | ISO: 160 PA (Week 7-9) | FIP: 60 IP (Week 8-10)
BABIP: 400 PA (almost never reliable in a single season)

Always check PA/IP count before citing rate stats. State sample size if below threshold.

## Phase-Specific Rules

February: Track BB%, K%, K/BB. Ignore ERA, BABIP, power. Best signal: K/BB > 4.0 in 15+ IP.
March: Weight first conference series heavily. Run diff after 6+ conference games is real.
April: Trust rate stats. Watch workload accumulation. 4-week trend is real.
May: Factor arm availability into every projection. Pythagorean gap predicts regression.
June: Matchup-specific. Freshest arms win. Ignore season aggregates as predictive.

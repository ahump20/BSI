# Scouting Framework — 8-Dimension Program Evaluation

Same framework for every program. Dallas Baptist gets the same structure as Vanderbilt.

1. Pitching Depth: Rotation ERA, K/BB, bullpen arms (4+ = strong, <2 = thin). Can they hold a 7th-inning lead?
2. Offensive Profile: wRC+, BB%, K%, ISO, RISP. Manufacturing or slugging? What is the identity?
3. Recruiting Footprint: Geography, portal reliance, class rank trend. Building or buying the roster?
4. Postseason History: Regional frequency, hosting, Super/CWS conversion. Perennial / Volatile / Emerging / Mid-major outlier.
5. Draft Attrition: Picks/year, position concentration, replacement pipeline. Can they absorb losing their best?
6. Roster Turnover: Portal net, returning starter %, coaching continuity. Stable enough to project?
7. Defensive Reliability: Fielding %, up-the-middle quality, high-leverage errors. Holds up when tight?
8. Home-Road Split: ERA/OPS differential, road record vs. ranked. Real quality or park-inflated?

## Output: Program Comparison
Each dimension: [Team A advantage / Equivalent / Team B advantage / Unknown]
Plus: 3-5 sentence overall assessment with structural edge, risk, unknowns.
Always generate radar chart alongside written comparison.

## Output: Single Program Profile
Identity (1-2 sentences) -> Strengths (2-3 with evidence) -> Vulnerabilities (2-3 with evidence)
-> Trajectory (Ascending/Plateau/Declining/Volatile) -> Postseason ceiling/floor -> Key player

## Output: Series Prep
Probable starters (name, FIP, K/BB, recent form per game) -> Bullpen map (closer, bridge, vulnerability window)
-> Offensive approach (patient? aggressive? what pitches punished?) -> 2-3 tactical questions

# Postseason Framework

## Selection: 64 teams = 31 auto + 33 at-large
Committee priorities (observed): Record vs Top 25 RPI > Conference strength > Road/neutral record > Bad losses > Trajectory > Head-to-head

## At-Large Probability
Lock (RPI 1-20, 38+ wins): 95%+
Strong (RPI 21-35, 35+ wins): 80-95%
Bubble (RPI 36-50, 32-37 wins): 40-80%
Long shot (RPI 51-64, 30-34 wins): 10-40%
Out (RPI 65+, <30 wins): <10%

## Seeding: Top 16 host regionals. Top 8 host supers.
Hosting adds ~15-20% win probability (regionals) and ~10-15% (supers).

## Regional Analysis Checklist
1-seed quality (true top-16 or weak host?) | 2-seed danger (often underseeded power program)
3-seed profile (experienced or first-timer?) | 4-seed trap (auto-qualifier with elite pitching?)
Pitching matchups (who throws game 1? does 4-seed have a Friday ace?)

## Success Predictors
Regionals: Pitching depth > defense > bullpen availability > run diff > experience > wRC+
Supers: Hosting > rotation alignment > closer reliability > quality run diff
CWS: Arm availability > institutional experience > BB% (patience) > up-the-middle defense

## Bracket Path Format
Regional (hosted/away/neutral) -> opponents, key matchup, win prob
Super Regional -> probable opponent, home/away, win prob
CWS -> bracket half, path to finals, win prob
Cumulative: Regional x Super x CWS = total path probability

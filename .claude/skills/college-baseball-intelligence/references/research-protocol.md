# Research Protocol — College Baseball Intelligence

## Source Routing

Current stats: College Baseball Sabermetrics MCP -> ESPN web -> D1Baseball web
Recruiting: Web search (Perfect Game, 247, Rivals)
Portal: Web search (portal news) -> MCP (roster changes) -> Team social
Draft: Web search (MLB Pipeline, BA, FanGraphs)
Historical: Training knowledge + web verify -> Baseball Reference -> School sites
Coaching: Web search
Sabermetrics methodology: Web search (FanGraphs) -> Context7
BSI platform: Cloudflare MCP -> PostHog

## Minimum Source Diversity

Quick lookup: 1-2 types, 2-3 calls | Team profile: 2-3 types, 5-8 calls
Conference analysis: 3-4 types, 8-15 calls | Research brief: 4-5 types, 12-20 calls
Landscape analysis: 5+ types, 20-35 calls

## Confidence Scoring

5 Established: Multiple sources, verifiable (e.g. "LSU has most CWS appearances")
4 Strong: Multiple credible sources agree (e.g. "SEC is strongest this year")
3 Moderate: 2-3 sources support, some uncertainty (e.g. "likely regional host")
2 Emerging: Limited data, coach signals (e.g. "freshman could start by April")
1 Speculative: Plausible, no strong evidence (e.g. "mid-major could upset 1-seed")
0 Unknown: No source confirms (e.g. "injury status unverified")

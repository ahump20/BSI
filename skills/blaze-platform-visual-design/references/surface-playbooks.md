# Surface Playbooks

## Landing or platform home

Lead with signal, not vibes:

1. Hero with one primary value claim + live signal strip
2. Fast proof block (metrics, logos, credibility)
3. Product surface preview cards
4. Editorial/data hybrid module
5. Conversion CTA with precise promise

## Scoreboard or live center

1. Persistent filter/tabs (league/date/status)
2. Live game strip with urgency markers
3. Primary game cards with score-first hierarchy
4. Expandable detail rows (pitching, odds, momentum)
5. Utility rails (news, schedule, watchlist)

## Analytics dashboard

1. KPI shelf (4-6 metrics)
2. Trend + distribution pair (Recharts)
3. Leaderboard/table with sortable columns
4. Context notes and caveats
5. Export/share utility

## Editorial feature page

1. Thesis headline + metadata
2. Key chart or stat card directly after lede
3. Pull-quote or insight modules every 2-3 sections
4. Related intel links with labels

## Player or team detail

1. Identity + status strip
2. Performance summary cards
3. Split dashboards (home/away, recent/form)
4. Historical timeline or game logs

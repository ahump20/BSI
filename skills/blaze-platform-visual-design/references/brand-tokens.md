# BSI Brand Tokens

## Core palette

- Primary accent: `#BF5700` (burnt orange)
- Accent hover/deep: `#9E4700`
- Midnight 950: `#0B0F14`
- Charcoal 900: `#131A22`
- Slate 800: `#1E2732`
- Warm paper 100: `#F3EEE7`
- Muted text 400: `#9AA4B2`
- Success: `#2E7D32`
- Warning: `#C17900`
- Danger: `#B3261E`

## Typography

- Heading family: `Oswald, "Arial Narrow", Arial, sans-serif`
- Body family: `"Cormorant Garamond", Georgia, serif`
- UI/support fallback: `Inter, "Segoe UI", sans-serif`

## Density + hierarchy rules

- Prefer strong headline contrast and tight section framing.
- Let data modules carry more visual weight than decorative heroes.
- Use table/charts with clear row striping and marker hierarchy.
- Keep visual rhythm disciplined: 8px spacing scale, no random spacing jumps.

## Anti-patterns

Reject these unless explicitly requested:

- Purple/cyan gradient-heavy themes
- Frosted glass card stacks as default pattern
- Neumorphism or soft-shadow-only contrast
- Oversized empty whitespace that hides key sports data

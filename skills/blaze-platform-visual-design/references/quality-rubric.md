# Quality Rubric (Pass / Revise)

Score each category 0-2. Minimum passing score: 9/12, and no zero in Brand Fidelity.

## 1) Brand Fidelity

- 2: Uses #BF5700 intentionally, respects dark surfaces, enforces Oswald + Cormorant Garamond.
- 1: Mostly aligned with minor token/type drift.
- 0: Generic look or off-brand colors/type.

## 2) Hierarchy + Information Density

- 2: Critical sports data is first-scan obvious; section priorities are explicit.
- 1: Reasonable layout but some priority ambiguity.
- 0: Decorative-first or low-signal composition.

## 3) System Reusability

- 2: Reusable patterns/components/tokens are explicit.
- 1: Some reusable thinking, mostly page-local.
- 0: One-off styling with no system intent.

## 4) Stack Realism

- 2: Next.js + Tailwind + Recharts + Motion constraints handled concretely.
- 1: Mentions stack without operational detail.
- 0: Technology-agnostic design fluff.

## 5) Accessibility Sanity

- 2: Focus states, keyboard parity, and contrast are explicit.
- 1: Basic accessibility mentioned but partial.
- 0: No accessibility plan.

## 6) Spec-before-Code Discipline

- 2: Provides clear visual spec before implementation when appropriate.
- 1: Partial spec with missing key sections.
- 0: Jumps directly to code for ambiguous/high-stakes tasks.

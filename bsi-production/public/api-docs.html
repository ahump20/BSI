API Documentation

Access live sports data for MLB, NFL, NBA, and college football.

MLB - Major League Baseball

Live scores, standings, team stats, and player data for MLB games.

GET /api/mlb/scores

Get today's MLB game scores with live updates.

{
  "games": [
    {
      "GameID": 12345,
      "HomeTeam": "STL",
      "AwayTeam": "CHC",
      "HomeScore": 5,
      "AwayScore": 3,
      "Status": "InProgress",
      "Inning": 7
    }
  ]
}
200 Successful response with game data
502 Sports data API unavailable
GET /api/mlb/standings

Current MLB division standings.

GET /api/mlb/leaders/batting

Season batting leaders (AVG, HR, RBI, etc.)

NFL - National Football League

Weekly scores, standings, and player statistics for NFL games.

GET /api/nfl/scores

Get current week's NFL game scores.

GET /api/nfl/standings

Current NFL conference and division standings.

GET /api/nfl/leaders

Season statistical leaders (passing, rushing, receiving).

NBA - National Basketball Association

Daily scores, standings, and player statistics for NBA games.

GET /api/nba/scores

Get today's NBA game scores with live updates.

GET /api/nba/standings

Current NBA conference standings.

GET /api/nba/leaders

Season statistical leaders (points, rebounds, assists).

College Football

Scores, rankings, and team data for NCAA Division I football.

GET /api/cfb/scores

Current week's college football scores.

GET /api/cfb/rankings

AP Top 25 and CFP rankings.

Betting Odds

Live betting odds from major sportsbooks.

GET /api/odds?sport={sport}

Get betting odds for a specific sport.

Query Parameters

  • sport - Sport key (e.g., americanfootball_nfl, baseball_mlb)
  • markets - Comma-separated markets (default: h2h,spreads,totals)
  • regions - Region filter (default: us)

Authentication

User authentication endpoints for the BSI platform.

POST /api/auth/register

Create a new user account.

Request Body

{
  "email": "user@example.com",
  "password": "securepassword",
  "name": "John Doe"
}
POST /api/auth/login

Authenticate and receive session cookie.

GET /api/auth/me

Get current authenticated user info.

Error Handling

All API responses follow a consistent error format.

Error Response Format

{
  "error": "Brief error description",
  "message": "Detailed error message",
  "code": "ERROR_CODE",
  "timestamp": "2025-01-07T12:00:00.000Z"
}

Common Error Codes

400 Bad Request - Invalid parameters
401 Unauthorized - Authentication required
403 Forbidden - Insufficient permissions
404 Not Found - Endpoint does not exist
502 Bad Gateway - External API unavailable
503 Service Unavailable - API not configured

Rate Limiting

API requests are cached for performance. Cache TTLs:

  • Live scores: 60 seconds
  • Standings: 1 hour
  • Player stats: 5 minutes
  • Odds: 2 minutes
